# RetailPulse-Powerbi-Dashboard
Interactive Power BI dashboard for supermarket sales analysis and business insights.

## Project Overview
RetailPulse Analytics is an interactive Power BI dashboard developed to analyze supermarket sales data and generate valuable business insights. The dashboard helps businesses monitor performance, identify trends, and make data-driven decisions.

---

## Objectives
- Analyze supermarket sales performance
- Monitor total revenue, profit, and quantity sold
- Identify top-performing product categories
- Compare branch performance
- Understand customer purchasing behavior
- Discover monthly sales trends

---

## Tools & Technologies
- Power BI
- Power Query
- DAX
- CSV Dataset

---

## Dashboard Features
- Interactive slicers and filters
- KPI cards for sales overview
- Sales distribution analysis using pie and donut charts
- Product category visualization using treemap
- Sales comparison using column charts
- Interactive data visualization

---

## Key Insights
- Food and beverages generated the highest sales revenue
- Certain branches outperformed others in total sales
- E-wallet payments were frequently used by customers
- Peak sales periods helped identify customer purchasing patterns

---

## Dataset Information
The dataset contains supermarket sales transactions including:
- Invoice ID
- Branch
- City
- Customer type
- Gender
- Product line
- Unit price
- Quantity
- Tax
- Total sales
- Payment method
- Customer rating
